총알의 종류는 많다.

조준탄 방향탄 n-way탄 원형탄 분열탄 유도탄 유도 레이저 미사일 가속탄 회전탄 회오리탄 등등

해당 탄들은 총알 클래스를 만들고 탄의 종류를 받아 행동을 달리하게 한다.


스테이지 enter -> 불릿 관련 layer 생성 -> 탄을 생성하는 class 초기화
스테이지 update -> 탄을 생성하는 class update -> 탄을 생성하여 불릿 관련 layer에 넣기

플레이어가 탄을 쏘면
플레이어 class update -> fire -> 탄을 생성하는 class로 넘어가서 생성 -> 해당 class 는 탄을 생성하여 불릿 관련 layer에 넣는다.

플레이어, 총알, 패턴, 총알을 패턴대로 만들어 발사

총알을 앞으로 발사하려면 필요한것 => 시작지점, 스피드, 각도
