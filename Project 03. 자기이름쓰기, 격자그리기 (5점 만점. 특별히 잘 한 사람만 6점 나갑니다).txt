4) recursive call (재귀함수 호출) 에 대해 조사하고 본문에 정리하라. 다음 내용이 포함되도록 한다.

- 일반적인 개념

- python 에서 활용 예

 

 

 

재귀적 호출 함수란는 재귀(再歸 : 원래의 자리로 되돌아가거나 되돌아옴) 함수입니다.

자기 자신을 다시 호출하는 함수로 특정 조건이 붙어있을 경우 함수가 그 조건에 따라서 반복 실행이 되도록 합니다.

예시로 간단하게 피보나치 수열을 구하는 재귀적 함수를 만들어 보았습니다.

 

def pibo(n):

if (n==1 or n==2):

return 1

elif (n<=0):

return 0

else:

return pibo(n-1)+pibo(n-2)

 

 

>>> num=pibo(10)

>>> num

55

 

이와 같이 재귀 함수를 사용하면 반복적인 계산을 [ 종결 조건 ]을 달리해주는 것으로 값을 계산 할 수 있습니다.

 

 

 

 

 

 

 

 

5) fractal graphic 에 대해 조사하고 본문에 정리하라. turtle graphics 와의 관계에 대해서도 조사해 보라. 예시를 몬문에 그림으로 붙여넣어도 좋다.

 

프랙탈(영어: fractal) 또는 프랙털은 일부 작은 조각이 전체와 비슷한 기하학적 형태를 말합니다. 이런 특징을 자기 유사성이라고 하며, 다시 말해 자기 유사성을 갖는 기하학적 구조를 프랙탈 구조라고 합니다. 프랙탈 구조는 자연물에서 뿐만 아니라 수학적 분석, 생태학적 계산, 위상 공간에 나타나는 운동모형 등 곳곳에서도 발견되어 자연이 가지는 기본적인 구조입니다. 불규칙하며 혼란스러워 보이는 현상을 배후에서 지배하는 규칙도 찾아낼 수 있습니다.

 

프랙탈은 수학적 도형으로도 연구되고 있습니다. 프랙탈 도형은 종종 컴퓨터 소프트웨어를 이용한 재귀적이거나 반복적인 작업에 의한 반복되는 패턴으로 만들어지는데, 대표적인 프랙탈 도형에는 망델브로 집합, 칸토어 집합, 시에르핀스키 삼각형, 페아노 곡선, 코흐 곡선 등이 있습니다.

 

(참고 사이트 : https://ko.wikipedia.org/wiki/%ED%94%84%EB%9E%99%ED%83%88)

 

파이썬에서 터틀 그래픽을 사용하면 몇가지 프렉탈을 재귀함수를 통해 그릴 수 있는데 그중 코흐 곡선을 예시로 들어보았습니다.

 

코흐 함수 예시

import turtle as t

 

t.penup()

t.goto(-500,0)

t.pendown()

 

def koch(length,n):

if (n == 0) :

t.forward(length)

else:

koch(length/3, n-1)

t.left(60)

koch(length/3, n-1)

t.right(120)

koch(length/3, n-1)

t.left(60)

koch(length/3, n-1)

 

 

t.speed(0)

koch(900,5)
