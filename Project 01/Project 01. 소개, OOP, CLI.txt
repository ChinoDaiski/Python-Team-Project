3.1 위 사이트의 설명을 따라하고 간단히 요약하라

명령 프롬프트 창(cmd)에서 할 수 있는 것이 적혀 있었습니다. Cd 명령어로 위치를 이동할 수 있고, mkdir 명령어로 폴더를 생성하고, rm 명령어로 해당 폴더를 지우는 내용이 있습니다.

 

3.2 본인이 가진 Command Line 방식의 숙련도에 대한 평가를 추가하라. ( 0 ~ 10 )

      GUI방식에 익숙해져 있다보니 Command Line 방식의 숙련도는 0입니다. 그래도 간단한 명령어로 해당 폴더에 들어갈 수 있다보니 코드면에선 압도적으로 편리함을 느낄 수 있었습니다.
